Extras

See also the Aquarium Rdocs.

Extras are tools that use Aquarium, but are not imported automatically when requiring aquarium.rb. Instead, you have to require aquarium/extras.rb explicitly.

V0.7.3
  • gem install aquarium
  • gem update aquarium
